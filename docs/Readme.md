# Ecommerce shop

# Project overview
In recent years, small offline businesses have been experiencing a decline in sales due to the rapid growth of online services. Modern consumers prefer to achieve their goals while spending a minimum amount of effort and time, which makes online solutions more attractive.

The purpose of this project is to develop a **web-based application using Spring Boot** that provides offline businesses with essential tools for:
- data analysis,
- accounting,
- sales management,
- optimization of business processes.

The application focuses on usability, scalability, and reliability.
